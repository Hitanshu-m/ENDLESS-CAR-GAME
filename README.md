# Endless Runner Car Game 🚗

A simple 2D endless runner car game built with Python and Kivy.

## About

This project is an image-based 2D car runner game where the player controls a car and avoids incoming obstacles. The game features continuous scrolling backgrounds, score tracking, obstacle spawning, pause functionality, and background music.

## Features

* 2D endless runner gameplay
* Three-lane road system
* Image-based graphics
* Infinite scrolling background
* Random obstacle generation
* Score counter
* Pause and resume support
* Background music
* Restart option after game over
* Keyboard and touch controls

## Controls

### Keyboard

* Left Arrow ← : Move Left
* Right Arrow → : Move Right
* Space Bar : Pause / Resume

### Touch Devices

* Tap left side of screen : Move Left
* Tap right side of screen : Move Right

## Requirements

* Python 3.x
* Kivy

Install dependencies:

pip install kivy

## Run the Game

python main.py

## Gameplay

The player must avoid obstacles appearing on the road. The longer you survive, the higher your score becomes. Colliding with an obstacle ends the game.
